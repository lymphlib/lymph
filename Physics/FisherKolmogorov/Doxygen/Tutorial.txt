\section sec_FKPP Fisher-Kolmogorov equation

This tutorial presents the solution of the Fisher--Kolmogorov equation using the PolyDG method implemented in the Physics/FisherKolmogorov folder.

The mathematical model reads
\f[
\frac{\partial c}{\partial t}
-
\nabla \cdot (\boldsymbol{\Sigma} \nabla c)
=
\alpha\,c(1-c)
+
I_{\mathrm{app}},
\qquad {\rm in} \;\Omega \times (0,T],
\f]
where \f$c\f$ denotes the scalar unknown variable, representing for instance a normalized population density or concentration field, \f$\boldsymbol{\Sigma}\f$ is the diffusion tensor, and \f$r\f$ is the reaction coefficient governing the growth dynamics.

The nonlinear reaction term
\f[
f(c)=\alpha\,c(1-c),
\f]
corresponds to the classical Fisher--Kolmogorov logistic law, combining linear growth for small values of \f$c\f$ with saturation effects as \f$c\f$ approaches the carrying capacity.

As a benchmark application of the presented PolyDG method, we consider a wave propagation problem on a two-dimensional domain.
The initial condition and the applied source term generate a traveling front that propagates through the computational domain according to the Fisher--Kolmogorov dynamics.

\section secTutFKPP1 Setup the input data

We set up these data in the script \b InputData/DataWavesFKPP.m.
In particular, we consider:
    - the computational domain \f$\Omega = (-1,3)\times(0,1)\f$ with Neumann boundary conditions; \n
    - isotropic diffusion with diffusion coefficient \f$d = 10^{-3}\f$; \n
    - the reaction parameter \f$\alpha = 1\f$; \n
    - the final simulation time \f$T = 1.0\f$; \n
    - the time integration step \f$\Delta t = 10^{-2}\f$; \n
    - a computational mesh with \f$N_{el} = 1000\f$ elements; \n
    - polynomial approximation degree equal to \f$3\f$; \n
    - penalty coefficient \f$C_\alpha = 10\f$; \n
    - Picard iterations for the nonlinear solver, with tolerance \f$10^{-10}\f$, maximum number of iterations equal to \f$1000\f$, and relaxation parameter equal to \f$1\f$; \n

We report here for completeness a sample of the script.

\verbatim
%% InputData/DataWavesFKPP.m
%% Convergence test for FK equation
...
Data.domain          = [-1 3 0 1];
Data.N               = 1000;
Data.meshfromfile    = true;
Data.foldername      = 'Meshes';
Data.VTKMeshFileName = 'WavesMesh.vtk';
Data.meshfileseq     = "FK_1000_el.mat";
...
%% Time discretization
Data.t0    = 0;
Data.T     = 1.00;
Data.dt    = 1e-2;
Data.theta    = 0.5;
%% Space discretization
Data.degree        = 3;		        % Polynomial degree
Data.penalty_coeff = 10;   			% Penalty coefficient
...
%% Adaptivity
Data.Adaptivity         = false;                % Flag of adaptivity                                             
...
\endverbatim

\section secTutFKPP2 Simulation run

We run the specific simulation using the script \b RunMainFKPP.m, 
which calls the main algorithm, contained in \b MainFunctions/MainFKPP.m.

\verbatim
%% RunMainFKPP.m
%% Initial Simulation Setup
MyPhysicsPath = pwd;
run('../SimulationSetup.m');
%% Input Data - Boundary conditions - Forcing term
DataWavesFKPP;
%% Mesh Generation
if Data.meshfromfile
    % Load an existing mesh
    Data.meshfile = fullfile(Data.foldername,Data.meshfileseq);
else
    [Data.meshfile] = MakeMeshMonodomain(Data,Data.N,Data.domain,Data.foldername,Data.meshfileseq,'P','laplacian');
end
%% Main 
[Error] = MainFKPP(Data,Setup)
\endverbatim

\section secAdaptiveFK l-adaptive framework
The \f$\ell\f$-adaptive strategy is driven by a local a-posteriori residual indicator, computed element-wise from the discrete solution, following \cite cangiani_posteriori_2023.
A detailed description of the estimator components is provided in \ref secAdaptiveHeat. Here, the indicator is extended to the time-dependent heat equation by accounting for both the current and previous time levels through the \f$\theta\f$-time discretization.
Given the discrete solutions \f$c_h^n\f$ and \f$c_h^{n-1}\f$, the local indicators associated with an element \f$K\f$ are defined as:

\subsection pAdaptiveSetupFK Setup Data 
We set up these data in the script \b InputData/DataWavesFKPP.m.

\verbatim
%% InputData/DataWavesFKPP.m 
%% Simulation for FKPP equation
...
%% Adaptivity
Data.Adaptivity         = true;                 % Flag of adaptivity                                             
Data.maxDegree          = 3;                    % Maximum polynomial degree
Data.AdaptFunc          = @(tau_r) floor(1 + (Data.maxDegree-1)*tanh(5*tau_r));     % Adaptivity function
Data.AdaptIts           = 2;                   % Maximum iterations of adaptivity
Data.AdaptivityStep     = 10; 
\endverbatim

The parameter \b Data.AdaptivityStep defines the frequency of the adaptive update during the time integration procedure.
