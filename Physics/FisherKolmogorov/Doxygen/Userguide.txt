\section secUGFK1 Setup and Configuration

  A single simulation in lymph use the following MATLAB scripts:
  - \b SimulationSetup.m   : setup and configuration
  - \b RunSetup.m   : run setup and configuration
  - \b RunMainFHN.m : run simulation 

The different proposed testcases are the following MATLAB scripts:
  - \b DataWavesFKPP.m: data input file
  - \b DataConvWavesFKPP.m : data input file for convergence with respect to the polynomial degree
  - \b DataConvTestDir.m : data input file for convergence with respect to mesh refinement
  - \b DatapAdaptiveWaveFKPP.m : data input file for convergence for adaptive testcase
 
   Other secondary scripts are:
  - \b RunhConvergenceFKPP.m: verification test for h convergence (see \ref secTutParConv)
  - \b RunpConvergenceFKPP.m: verification test for p convergence (see \ref secTutParConv)

\subsection subUGFFK1 Simulation Setup

The users can configure or customize the code for their specific needs.
In particular, \b RunSetup.m defines the  simulation setup and contains the following fields:
\verbatim
- isPlotMesh    : 1/0 to plot the polygonal mesh
- isSaveVTKMesh : 1/0 to save the polygonal mesh in the VTK format
- isPlotSolution: 1/0 to visualize the numerical solution in Matlab
- isSaveSolution: 1/0 to save the solution in the .mat format
- OutFolder     : folder name for the output (e.g. Output)
- isSaveCSV     : 1/0 to save the solution in the .csv format
- isSaveVTK     : 1/0 to save the solution in the .vtk format
- OutFolderVTK  : folder name for the VTK-output
- isError       : 1/0 to compute the error (e.g. OutputVTK)
- isParallel    : 1/0 for parallel implementation
- numCores      : number of cores for parallel implementation
\endverbatim

\subsection subUGFFK2 Input data files

To define the input data for the specific simulation you can use a MATLAB script (e.g. \b InputData/DataWavesFKPP.m).
You can set up the number of the element mesh, the polynomial approximation degree, the penalty constant, final time, time step, time integration scheme, 
boundary conditions, forcing terms and analystical solution (if needed). 

 \verbatim
%% DataWavesFKPP.m
%% Convergence test for FKPP equation
Data.name = 'WavesTest';
Data.LabEl = {'K'};     % Element labels
Data.TagEl = { 1 };     % Element tags
Data.TagBc = {[2 3 4 5]};  % Boundary tags
Data.LabBc = {'NNNN'};  % (D)irichlet/(N)eumann/(A)bsorbing
Data.TagApplyBCs = 0;      % Skip the assembling of BCs if not necessary
%% Geometrical properties
Data.domain          = [-1 1 -1 1];
Data.N               = 180; 
Data.meshfromfile    = true;
Data.foldername      = 'InputMesh';
Data.VTKMeshFileName = 'WavesMesh.vtk';
Data.meshfileseq     = "FHN_180_el.mat";
...
%% Material properties 
Data.D_ext       = {@(x,y,t) 1+0.*x};  % External diffusion
Data.D_axn       = {@(x,y,t) 0.*x};       % Axonal diffusion
Data.Cm          = {@(x,y,t) 1 + 0.*x}; 
Data.Chi         = {@(x,y,t) 1 + 0.*x};
Data.k       = {@(x,y,t) 1 + 0.*x};
Data.epsilon = {@(x,y,t) 1 + 0.*x};
Data.gamma   = {@(x,y,t) 2 + 0.*x};
Data.beta    = {@(x,y,t) 1 + 0.*x};
Data.a       = {@(x,y,t) 0.5 + 0.*x};
...
%% Discretization properties
%% Time discretization
Data.t0    = 0;
Data.T     = 2.00;
Data.dt    = 1e-2;
Data.theta    = 1;
%% Space discretization
Data.degree        = 3;		        % Polynomial degree
Data.penalty_coeff = 10;   			% Penalty coefficient
...
%% Adaptivity
Data.Adaptivity         = false;                % Flag of adaptivity                                             
...
\end{lstlisting}
\endverbatim

\subsection subUGFK3 Run a simulation
With the file \b RunMainFHN.m you can run the specific simulation. 

\verbatim
%% RunMainFHN.m
%% Initial Simulation Setup
MyPhysicsPath = pwd;
run('../SimulationSetup.m');
%% Input Data - Boundary conditions - Forcing term
DataTestCaseFHN
%DataSpiral;
%% Mesh Generation
if Data.meshfromfile
    % Load an existing mesh
    Data.meshfile = fullfile(Data.foldername,Data.meshfileseq);
else
    %[Data.meshfile] = MakeMeshMonodomain(Data,Data.N,Data.domain,Data.foldername,Data.meshfileseq,'C','laplacian');
    [Data.meshfile] = MakeMeshMonodomain(Data,Data.N,Data.domain,Data.foldername,Data.meshfileseq,'P','laplacian');
end
%% Main 
[Error] = MainFHN(Data,Setup)
\endverbatim
